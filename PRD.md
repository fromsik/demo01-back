# PRD.md - TimeBox Planner 제품 북극성

## 1) 제품 한 줄 정의
TimeBox Planner는 **하루의 핵심 목표(1~5개)**와 **10분 단위 시간 블록 계획**을 빠르게 작성/저장/재사용하는 타임박싱 서비스다.

## 2) 플랫폼 원칙
- **웹은 로그인 필수** (Google/Kakao/Naver OAuth).
- **앱은 로컬 모드 우선** (로그인 없이 기기 내부 저장, 서버 전송 없음).
- 서버 저장 데이터는 사용자 소유권을 엄격히 분리한다.

## 3) 백엔드 핵심 정책
- 목표는 1~5개만 허용.
- 시간 블록은 HH:MM 형식, 10분 단위만 허용.
- 시작 시간 < 종료 시간.
- 동일 일별 계획 내 시간 블록 겹침 금지.
- 템플릿 수정은 과거 일별 계획에 자동 반영하지 않음.

## 4) MVP 백엔드 범위
- 인증 API: `/api/auth/*`
- 템플릿 API: CRUD + 복사 + 기본 템플릿 지정
- 일별 계획 API: CRUD + 템플릿 기반 생성 + 날짜 조회
- 로그인 사용자별 데이터 분리/검증

## 5) 제외 범위(후속)
- 웹/앱 실시간 동기화
- 팀 공유, AI 추천, 알림, 캘린더 연동, 통계
